# 카카오 챗봇 연결 절차

## 1. 채널 준비

카카오톡 채널 관리자센터에서 학교 관측용 채널을 만듭니다.

권장 채널명:

```text
광곽현웅봇
```

## 2. 챗봇 기본 메뉴

챗봇 관리자센터에서 아래 발화를 기본 블록 또는 웰컴 블록에 연결합니다.

```text
오늘 관측 가능?
장비가 이상해요
뭘 보면 좋을까요?
관측 기록하기
안전/비상 안내
```

FITS 기록 예시 발화:

```text
기록: fits=20260423_M42_R_001.fits, 대상=M42, 필터=R, 노출=60s, 메모=구름 조금
```

## 3. 스킬 서버 연결

로컬 개발 서버의 카카오 스킬 엔드포인트는 다음입니다.

```text
POST /skill
```

카카오에 연결하려면 공개 HTTPS 주소가 필요합니다.

테스트 단계에서는 터널링 도구를 사용할 수 있고, 운영 단계에서는 클라우드 서버나 학교 서버에 배포하는 편이 좋습니다.

## 4. 검수 전 확인

- 태양 관측 위험 안내가 우선 노출되는지 확인
- 학생 개인정보를 저장하지 않는지 확인
- 잘못된 답을 할 수 있는 자유형 AI 답변은 교사 검토 후 활성화
- 관측 기록 저장 위치와 접근 권한 확인
